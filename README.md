# Data-Analysis-Project Portfolio

This repository contains my completed Data Analysis projects and assessments completed during my Data Analysis training with Torbita Computer Training School, Abuja, Nigeria. The projects demonstrate practical application of data analytics techniques using Microsoft Excel, SQL, Power BI, and Python to solve real-world business problems.

The repository includes solutions to all assigned tasks, project files, dashboards, SQL scripts, Python notebooks, and supporting documents.
